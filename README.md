# Just Another Farming Sim

play at https://christian-post.github.io/just-another-farming-sim/


# Controls 

| function        | key      | button   |
| --------------- | -------- | -------- |
| move            | cursors  | D-pad    |
| interact        | W        | X        |
| open/close menu | E        | Y        |
| item 1          | A        | A        |
| item 2          | S        | B        |
