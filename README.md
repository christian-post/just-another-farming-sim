# Just Another Farming Sim

Play the current version at https://christian-post.github.io/just-another-farming-sim/


# Controls 

Currently supported gamepads: XBOX 

| function             | keyboard | gamepad  |
| ---------------      | -------- | -------- |
| move                 | cursors  | D-pad    |
| interact             | W        | X        |
| inventory            | E        | Y        |
| use item 1           | A        | A        |
| use item 2           | S        | B        |
| menu                 | ESC      | BACK     |
